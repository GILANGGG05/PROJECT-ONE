\## 📋 FITUR LENGKAP

\- ✅ \*\*Order tanpa login\*\* - Langsung isi ID game, bayar via QRIS

\- ✅ \*\*Login/Register\*\* - Pakai akun untuk lihat riwayat

\- ✅ \*\*Merge Order Guest\*\* - Order sebelum daftar otomatis masuk ke akun

\- ✅ \*\*QRIS Payment\*\* - Tersedia QRIS statis (bisa upgrade ke dinamis)

\- ✅ \*\*Cek Pesanan\*\* - Lacak status tanpa login (pakai Order ID + WA)

\- ✅ \*\*Dashboard User\*\* - Lihat semua riwayat top up

\- ✅ \*\*12+ Game Populer\*\* - ML, FF, PUBG, Genshin, dll

\- ✅ \*\*Responsive Design\*\* - Bisa diakses dari HP/Tablet/Desktop
